# OCCOPSEP
This program changes the execution policy of PowerShell to execute local scripts and untrusted cloud scripts.And you don't have to type your own hard-to-understand PowerShell.A series of operations can be completed directly only by selecting the input number.

It is important to note that the user must be guaranteed or granted administrator privileges.Otherwise, the program will refuse to operate.
At the same time, you should also pay attention to some extreme execution strategies, which may cause some problems with your computer.
